🚀 개발 명령어

pnpm install              # 의존성 설치
pnpm dev                  # 모든 앱 개발 모드 실행
pnpm build                # 모든 앱 빌드
pnpm type-check          # TypeScript 타입 체킹
pnpm lint                # 린팅
pnpm test                # 테스트 실행

🌐 로컬 포트
• 프론트엔드: http://localhost:3000  (Next.js)
• 백엔드:     http://localhost:3001  (NestJS)
• WebSocket:  ws://localhost:3001    (Socket.io)
